# 实用脚本

这个目录包含一些实用的维护脚本。

## 可用脚本

### 数据管理

- `clear_memory.bat` / `clear_memory.py` - 清空记忆数据库（保留群友信息）
- `clean_data.bat` - 清理所有数据（包括日志、数据库）

### 配置管理

- `check_config.bat` - 检查配置文件是否正确

### 日志查看

- `watch_log.bat` - 实时查看 Bot 日志

## 使用方法

直接双击运行对应的 `.bat` 文件即可。

## 注意事项

- 清理数据操作不可恢复，请谨慎使用
- 建议在清理前备份重要数据
