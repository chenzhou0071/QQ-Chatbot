# 联网搜索功能说明

## 功能介绍

舟舟现在支持联网搜索功能，基于**通义 Web-Search MCP 服务**，可以实时获取互联网信息。

## 触发条件

当消息中包含以下关键词时，会自动触发联网搜索：

- **天气相关**：天气、气温、温度、下雨、晴天、阴天
- **时间相关**：时间、几点、现在、日期、星期
- **新闻相关**：新闻、热搜、最新、今天
- **娱乐相关**：笑话、段子
- **金融相关**：股票、金价、油价、汇率、美元、人民币

## 配置方法

### 1. 获取 API Key

访问 [阿里云百炼](https://bailian.console.aliyun.com/) 获取 `DASHSCOPE_API_KEY`

### 2. 配置环境变量

在 `config/.env` 文件中添加：

```env
DASHSCOPE_API_KEY=your_dashscope_api_key_here
```

### 3. 重启机器人

配置完成后重启机器人即可生效。

## 使用示例

**用户**：@舟舟 今天北京天气怎么样？  
**舟舟**：（联网搜索后）嗯...今天北京是晴天呢，气温大概15度左右☀️

**用户**：舟舟，现在几点了？  
**舟舟**：（联网搜索后）现在是...下午3点半呢

**用户**：@舟舟 讲个笑话  
**舟舟**：（联网搜索后）那个...我找到一个笑话...（害羞地讲笑话）

## 特点

- ✅ 自动判断是否需要联网搜索
- ✅ 搜索结果会融入舟舟的人设回复
- ✅ 保持腼腆害羞的语气
- ✅ 支持实时信息查询
- ✅ 免费额度：每月 2000 次

## 限制

- 单用户 5 QPS
- 每月 2000 次免费额度
- 额度用尽后自动停止

## 注意事项

- 如果未配置 `DASHSCOPE_API_KEY`，联网搜索功能将不可用，但不影响其他功能
- 搜索结果会作为上下文提供给 AI，AI 会基于人设进行回复
- 搜索超时时间为 10 秒
