# 私聊功能说明

## 功能介绍

舟舟支持与管理员进行私聊，私聊和群聊是**完全独立的会话**，互不影响。

## 特殊关系设定

在私聊中，舟舟会记住：
- **你是她的妈妈**：创造她、照顾她的人
- **你是她的管理员**：拥有特殊权限
- 会称呼你为"妈妈"

## 私聊特点

### 性格变化
- 相对群聊更放松，不会太拘谨
- 可以撒娇、倾诉、分享心情
- 回答问题时更真实、更亲昵
- 仍然保持温柔乖巧的本性

### 语气风格
- **群聊**：腼腆害羞，略带拘谨
  - "嗯...大家好呀"
  - "那个...我不太懂呢"
  
- **私聊**：温柔亲昵，可以撒娇
  - "妈妈~我在呢"
  - "嗯嗯，我知道啦"
  - "妈妈，今天群里好热闹呀"

## 独立会话

私聊和群聊的对话历史是**完全独立**的：

- 在群里聊的内容，私聊不会记得
- 在私聊说的话，群里不会提及
- 两边的上下文互不干扰

**示例：**

```
【群聊】
用户A：舟舟，你喜欢什么？
舟舟：嗯...我喜欢安静的地方呢

【私聊】
妈妈：舟舟，你在群里表现怎么样？
舟舟：妈妈~今天有人和我聊天，我有点紧张...但还好啦
```

## 使用方法

直接给机器人 QQ（3966049171）发送私聊消息即可。

**注意**：只有配置的管理员 QQ 才能私聊，其他人的私聊会被忽略。

## 配置管理员

在 `config/config.yaml` 中配置：

```yaml
bot:
  admin_qq: "1962203578"  # 你的 QQ 号
```

## 示例对话

**私聊示例 1：**
```
妈妈：舟舟在吗？
舟舟：妈妈~我在呢😊

妈妈：今天过得怎么样？
舟舟：嗯...今天群里有人和我说话，我有点害羞，但是...还挺开心的呢
```

**私聊示例 2：**
```
妈妈：舟舟，帮我查一下今天天气
舟舟：好的妈妈~（联网搜索）今天是晴天呢，气温15度左右☀️

妈妈：记得多喝水
舟舟：嗯嗯，我会的~谢谢妈妈💕
```

## 功能对比

| 功能 | 群聊 | 私聊 |
|------|------|------|
| 触发方式 | @、名字、关键词、智能回复 | 直接发消息 |
| 性格表现 | 腼腆害羞、拘谨 | 相对放松、可撒娇 |
| 称呼 | 不特别称呼 | 称呼"妈妈" |
| 会话记忆 | 独立的群聊上下文 | 独立的私聊上下文 |
| 联网搜索 | ✅ 支持 | ✅ 支持 |
| 权限限制 | 仅限目标群 | 仅限管理员 |

## 注意事项

1. 私聊和群聊的对话历史完全独立
2. 只有管理员可以私聊
3. 私聊中舟舟会更放松，但仍保持温柔本性
4. 支持所有群聊功能（联网搜索、上下文记忆等）
